To get started on Project Gestures:
  • Upload the Project Gestures sketch to an Arduino Nano BLE Sense board
  • Connect to the board using the Bluetooth button below.
  • Wave the wand to make gestures. They'll be displayed on the left.
  • The prediction class will be displayed next to it. NOTE: ["0: Swipe up", "1: Swipe down", "2: Hand wave", "3: Clockwise", "4: Counter-clockwise"]
  • The following API calls will be made based on the prediction class:
    • 0: Swipe up -> Turn on Light
    • 1: Swipe down -> Turn off Light
    • 2: Hand Wave -> TBD
    • 3: Clockwise -> Turn on and brighten
    • 4: Counter-Clockwise -> Turn on and dim
